# yad2-back
This is back-end for Yad-2 (front) React app.<br>

Server architecture:<br>
Server design keeps clean single task coding. tasks are logically distributed to controllers which can be easily reused.<br>

Data is save in nonraltioanl db.<br>
Using mongo/mongoose to store textual data.<br>
Using aws bucket to store images.<br>
Using JWT to authenticate users.<br>
